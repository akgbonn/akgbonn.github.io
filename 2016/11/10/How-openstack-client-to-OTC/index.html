Obtaining and Installing OpenStack Clients

Install OpenStack Client. The version requirements are as follows:

  • Nova Client: 2.22.0
  • Cinder Client: 1.1.1
  • Glance Client: 0.15.0
  • Neutron Client: 2.3.11
  • Keystone Client: 1.2.0

Obtaining OpenStack Clients

Nova Client https://github.com/openstack/python-novaclient.git
Glance Client https://github.com/openstack/python-glanceclient.git
Neutron Client https://github.com/openstack/python-neutronclient.git
Cinder Client https://github.com/openstack/python-cinderclient.git
Keystone Client https://github.com/openstack/python-keystoneclient.git

Configuring OpenStack Client

The OpenStack APIs provided by the Open Telekom Cloud system use HTTPS. Therefore, you need to add parameter –insecure when you use OpenStack Client to visit the system.
If you require to use a command without parameter –insecure, see A.5 Configuring the CLI Not Containing insecure.
